# AI Rehab Tracker

A web-based rehabilitation tracking app for post-surgery patients (ACL, meniscus, orthopedic rehab).

## Problem
Patients recovering from orthopedic surgery struggle with:
- Tracking exercises
- Measuring pain, ROM, and swelling
- Staying consistent without constant physio supervision

## Solution
An AI-assisted rehab tracker that:
- Logs daily exercises
- Tracks pain, swelling, flexion/extension
- Provides recovery insights and warnings
- Helps patients stay compliant and motivated

## Tech Stack (Planned)
- React
- Node.js / Express
- AI (OpenAI or rule-based model)
- Charting (Recharts)
